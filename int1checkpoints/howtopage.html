The Navigation Bar

In order to make the navigation bar, I was going to need a way to keep all of the navigation options on one line. It doesn't look particularly good when your navigation bar is vertical. In order to keep the navigation bar options separate, I would need them to be separate items. One way to keep items separate on a list,( a navigation bar is really just a list of links) is to make them list items.

A literal list of links will turn out looking like this:




<ul>

<a href="moreaboutme.html"><li>More About Me</li></a>

<a href="listoftags.html"><li>What I've Learned About Coding</li></a>

<a href="howtopage.html"><li>How I Made the Homepage and Both Subpages</li>

</ul>

Unfortunately, if I use this code, this is what it will look like in the webpage:

  • More About Me
    •
  • What I've Learned About Coding
    •
  • How I Made the Homepage and Both Subpages
    •

In order to fix the navigation bar and make it actually look like the one you see at the top of this page, I would have to add some CSS code. In order to change the navbar to what you see at the top of this page, I would have to change the font color, the backgrund color, add a border, delete the bullet points, put all three list items on one line, put a little bit of spacing in between them, and get rid of the text decoration.

To start, I attached the 'navbar' class to the list so that my CSS would only affect that specific list
and linked my homepage to a CSS file. I used this CSS code to accomplish my said goals:





  • .navbar {
  • background-color:black;
  • text-align:center;
  • }
  • .navbar a {
  • display:inline-block;
  • color:white;
  • border:2px solid white;
  • text-decoration: none;
  • padding:10px;
  • }
  • .navbar li {
  • list-style: none;
  • }

Since I wanted a black background for the navbar, I gave all things that have the 'navbar' class the declaration.
For this, I used the background-color declaration, which changes the background color. My code looked like this:

  • .navbar {
  • background-color:black;
    •
  • }

Next I centered all things with the 'navbar' class, so that my navigation bar was easier to notice.
For this, I used the text-align declaration, which changes the alignment of the text color. My code looked like this:

  • .navbar {
  • background-color:black;
    •
  • text-align:center;
    •
  • }

After that I put all links with the 'navbar' class in a straight line, so that my navigation bar actually a straight line,
not a box. For this, I used the display declaration, which changes the layout. My code looked like this:



  • .navbar {
  • background-color:black;
    •
  • text-align:center;
    •
  • }
  • .navbar a {
  • display:inline-block;
  • }

Next I made all links with the 'navbar' class have a white font so that my navigation bar more visible.
In order to do this I used the color declaration, which changes the font color. My code looked like this:



  • .navbar {
  • background-color:black;
    •
  • text-align:center;
    •
  • }
  • .navbar a {
  • display:inline-block;
  • color: white;
  • }

After that, I added a border to all links with the 'navbar' class, so that they were better separated.
In order to do this I used the border declaration, which changes the border size. My code looked like this:



  • .navbar {
  • background-color:black;
    •
  • text-align:center;
    •
  • }
  • .navbar a {
  • display:inline-block;
  • color: white;
  • border:2px solid white;
  • }

Next I got rid of the underlining of all links with the 'navbar' class, so that they weren't ugly(I don't like how that looks).
In order to do this I used the text-decoration declaration, which changes the decoration of the text. My code looked like this:



  • .navbar {
  • background-color:black;
    •
  • text-align:center;
    •
  • }
  • .navbar a {
  • display:inline-block;
  • color: white;
  • border:2px solid white;
  • text-decoration:none;
  • }

Next, I separated the all of the links with the 'navbar' class even more, so that the options weren't smushed together.
In order to do this I used the padding declaration, which changes the space between the object(including the edge of the screen)
and the object the padding is on. At that point, my code looked like this:



  • .navbar {
  • background-color:black;
    •
  • text-align:center;
    •
  • }
  • .navbar a {
  • display:inline-block;
  • color: white;
  • border:2px solid white;
  • text-decoration:none;
  • padding:10px;
  • }

Finally, I deleted all of the bullet points on the list items with the 'navbar' class, mostly because navigation bars don't
have bullet points. In order to do this I used the list-style declaration, which sets all properties of a list in one
declaration. At that point, my code looked like this:



  • .navbar {
  • background-color:black;
    •
  • text-align:center;
    •
  • }
  • .navbar a {
  • display:inline-block;
  • color: white;
  • border:2px solid white;
  • text-decoration:none;
  • padding:10px;
  • }
  • .navbar li {
  • list-style:none;
  • }

The Background

Technically speaking, the background color is white. Despite this, when you are looking at my home page, you will notice large portions where the background color changes to black and the font color changes to white. I decided that I would do this to separate the two segments of my homepage: "About Me" & "What Have I Learned?". In order to make these gigantic blocks where the background and text color change, I used this html code:
  • <div class="aboutme">
  • The actual content
  • </div>

I wrap this div around the content, as shown above.The whole
purpose of this is to link the div to CSS code. The CSS file I linked the'aboutme' div to
contains this code:

  • .aboutme {
  • padding:15px;
    •
  • background-color:black;
    •
  • color:white;
    •
  • text-align:right;
    •
  • }

In this CSS, the padding declaration adds a little space between the content and other objects on the page(including the ede of the screen), the background-color declaration chages the background color to my desired color(black), the color declaration changes the text color to my desired color(white), and the text-align declaration puts the text on my desired side of the webpage(right).

For my second topic, "What Have I Learned?", I used the same div with a different name and a singular difference: the alignment. Instead of the text alignment being set to "right", it is set to "center".
